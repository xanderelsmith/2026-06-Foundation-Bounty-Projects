# Code of Conduct

## 🌍 Our Pledge

We are committed to creating an open, welcoming, and respectful environment for everyone participating in this bounty program, regardless of:
- Experience level
- Gender identity and expression
- Sexual orientation
- Disability
- Personal appearance
- Body size
- Race
- Ethnicity
- Nationality
- Religion
- Technical background

We value constructive collaboration, professionalism, and mutual respect.

---

# 🤝 Expected Behavior

Participants, reviewers, and maintainers are expected to:
- Be respectful and inclusive
- Provide constructive feedback
- Communicate professionally
- Respect differing opinions and viewpoints
- Focus on what is best for the community and ecosystem
- Accept feedback gracefully
- Help create a positive and collaborative environment

---

# 🚫 Unacceptable Behavior

The following behaviors will not be tolerated:
- Harassment or discrimination of any kind
- Personal attacks or insults
- Trolling or inflammatory comments
- Threats or intimidation
- Publishing private information without consent
- Plagiarism or fraudulent submissions
- Spam or malicious activity
- Any conduct that negatively impacts the community experience

---

# 📝 Pull Request & Review Etiquette

When participating in the application or review process:
- Keep discussions relevant to the project
- Respond respectfully to feedback
- Avoid unnecessary hostility or arguments
- Reviewers should provide fair and objective evaluations
- Applicants should communicate changes clearly

---

# ⚖️ Enforcement Responsibilities

Repository maintainers and program reviewers are responsible for:
- Clarifying acceptable behavior
- Reviewing reported incidents
- Taking appropriate and fair corrective action

Actions may include:
- Warning the participant
- Closing pull requests
- Rejecting applications
- Temporary or permanent bans from participation

---

# 🚨 Reporting Issues

If you experience or witness unacceptable behavior, please contact the repository maintainers directly.

All reports will be reviewed confidentially and handled appropriately.

---

# 📌 Scope

This Code of Conduct applies to:
- Pull requests
- Issues
- Discussions
- Review comments
- Community interactions related to this bounty program

---

# ❤️ Community First

This program exists to encourage innovation, collaboration, and ecosystem growth.

By participating, you agree to uphold these standards and help create a welcoming environment for everyone.
